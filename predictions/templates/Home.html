About the Idea

This project utilizes Indian prison data from 2001 to 2013 to analyze and predict trends in crime rates and demographic patterns of inmates based on variables like gender, age, region, and trial status. We first do exploratory data analysis (EDA) and data cleaning, then use machine learning algorithms to build predictive models. Insights aim to optimize resource allocation and recommend policy interventions.

In this project, we employ machine learning and statistical models to identify trends in the status of the prison inmates (convicted vs undertrial) with respect to the crime committed, assess the impact of demographic factors on inmate profiles, and predict future prison resource allocations using historical data. The insights support policy reforms, targeting reduced recidivism, equitable justice, and improved prison management.

Project By: Apoorv, Jainam, Kalash, and Zubair
